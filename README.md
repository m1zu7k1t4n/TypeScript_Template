# TypeScript Template

NodeVersion: 10.14.2

## DevTools

- vscode
- npm
- npm-run

## command

`npm run build`
Build src_dir, output to build_dir

`npm run watch`
Monitor , build changes

`npm test`
Testing unit_test in test_dir

`npm run release`
Generate One-Package in build_dir
so Executeing `npm install (One-Package).tgz` ,
Installed this Project.
